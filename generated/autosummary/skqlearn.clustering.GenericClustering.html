GenericClustering¶

class skqlearn.clustering.GenericClustering(n_clusters: int, max_iterations: int, random_state: Union[int, numpy.random.mtrand.RandomState, None], distance_calculation_method: str)¶

Bases: abc.ABC

Generic clustering algorithm used as base for K-Means and K-Medians.

Through a series of epochs, the algorithm assigns each input sample/vector to the closest centroid. Then, the centroids are recalculated based on the metric implemented in each specific algorithm. The distance used is the Euclidean or \(L^2\) norm.

Construct a GenericClustering object.

Parameters

  • n_clusters (int) – Number of clusters to form and centroids to generate.

  • max_iterations (int) – Max number of iterations of the algorithm.

  • random_state (int, RandomState or None) – Determines random number generation for centroid initialization. If provided as an int, it will be used as seed to make the randomness deterministic.

  • distance_calculation_method ({'classic', 'quantum'}) –

    The distance calculation method:

    • ’classic’: Classically calculated using np.linalg.norm.

    • ’quantum’: Quantum distance estimation will be used.
