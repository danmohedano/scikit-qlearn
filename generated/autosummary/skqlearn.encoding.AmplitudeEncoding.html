AmplitudeEncoding¶

class skqlearn.encoding.AmplitudeEncoding(degree: int = 1)¶

Bases: skqlearn.encoding.base_encoding.Encoding

Amplitude encoding method. [SP18]

In amplitude encoding, each component of the input vector \(\boldsymbol{x} \in \mathbb{R}^N\) is mapped to an amplitude of the quantum state, defining the encoding feature map as:

\[\phi:\boldsymbol{x}\rightarrow\left|\psi_\boldsymbol{x}\right>= \sum_{i=1}^{N}\frac{1}{|\boldsymbol{x}|}x_i\left|i-1\right>\]

In order to represent a valid quantum state the amount of amplitudes, and therefore, the dimension of the vectors must be a power of 2, \(N=2^n\). If they are not, they will be padded with zeros at the end.

Therefore, the kernel defined by the inner product is the linear kernel after applying a correction of \(|\boldsymbol{x}||\boldsymbol{x'}|\):

\[k(\boldsymbol{x}, \boldsymbol{x'}) = \left<\psi_{\boldsymbol{x}}| \psi_{\boldsymbol{x'}}\right> = \frac{1}{|\boldsymbol{x}| |\boldsymbol{x'}|}\boldsymbol{x}^T\boldsymbol{x'}\]

By, instead, mapping the input to \(d\) copies of an amplitude encoded quantum state, a polynomial kernel can be defined after correction:

\[\phi:\boldsymbol{x}\rightarrow\left|\psi_\boldsymbol{x}\right> ^{\bigotimes d}\]
\[k(\boldsymbol{x}, \boldsymbol{x'}) = \left<\psi_{\boldsymbol{x}}| \psi_{\boldsymbol{x'}}\right> \bigotimes ... \bigotimes \left<\psi_{\boldsymbol{x}}|\psi_{\boldsymbol{x'}}\right> = \left(\frac{1}{|\boldsymbol{x}||\boldsymbol{x'}|}\boldsymbol{x}^T \boldsymbol{x'}\right)^d\]

A dataset can be encoded by concatenating all the input vectors.

degree¶

Desired degree of the polynomial kernel defined by the encoding. In turn, it defines the amount of copies of each input vector that are encoded into the quantum state.

Type

int

Parameters

encoding(x: numpy.ndarray) → numpy.ndarray¶

Application of amplitude encoding to the input.

Parameters

x (numpy.ndarray of shape (n_features) or (n_samples, n_features)) –

Input. This can be a single sample of shape (n_features,) or a dataset of shape (n_samples, n_features).

Note

The input must be normalized in order to define a valid quantum state. If it is not, the method will normalize the input.

Returns

Quantum state described as an amplitude vector with the amount of copies of the state indicated in the constructor. If a dataset is provided, the states are concatenated.

Return type

numpy.ndarray

Raises

ValueError – If an invalid input type or an empty vector (all components have value 0) is provided.

Examples

>>> a = np.array([0.5, 0.5, 0.5, 0.5])
>>> AmplitudeEncoding().encoding(a)
array([0.5, 0.5, 0.5, 0.5])

>>> a = np.array([0.0, 1.0, 0.0])
>>> AmplitudeEncoding().encoding(a)
array([0., 1., 0., 0.])
