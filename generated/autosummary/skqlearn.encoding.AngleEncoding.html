AngleEncoding¶

class skqlearn.encoding.AngleEncoding¶

Bases: skqlearn.encoding.base_encoding.Encoding

Angle encoding method. [LC20]

In angle encoding, each component/feature of the input vector \(\boldsymbol{x} \in \mathbb{R}^N\) is mapped to a qubit, defining the encoding feature map as:

\[\phi:\boldsymbol{x}\rightarrow\left|\psi_\boldsymbol{x}\right>= \bigotimes_{i=1}^{N}\cos{x_i}\left|0\right>+\sin{x_i}\left|1\right>\]

Because of the encoding feature map, the resulting quantum state is correctly normalized and therefore valid, as \(\cos{x}^2+\sin{x}^2=1\).

The kernel defined by the inner product is a cosine kernel:

\[k(\boldsymbol{x}, \boldsymbol{x'}) = \left<\psi_{\boldsymbol{x}}| \psi_{\boldsymbol{x'}}\right> = \prod_{i=1}^{N}\sin{x_i}\sin{x'_i} + \cos{x_i} \cos{x'_i}=\prod_{i=1}^{N}\cos{(x_i-x'_i)}\]

encoding(x: numpy.ndarray) → numpy.ndarray¶

Application of angle encoding to the input.

In angle encoding, each feature of the input vector is encoded into a qubit.

Parameters

x (np.ndarray of shape (n_features,)) – Input vector.

Returns

Quantum state described with an amplitude vector.

Return type

np.ndarray

Raises

ValueError – When an invalid input is provided.

Examples

>>> a = np.array([0.0])
>>> AngleEncoding().encoding(a)
array([1.0, 0.0])

>>> a = np.array([np.pi / 2, 0])
>>> AngleEncoding().encoding(a)
array([6.123234e-17, 0.000000e+00, 1.000000e+00, 0.000000e+00])
