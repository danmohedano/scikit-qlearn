BasisEncoding¶

class skqlearn.encoding.BasisEncoding¶

Bases: skqlearn.encoding.base_encoding.Encoding

Basis encoding method. [SP18]

In basis encoding, each classical bit is mapped into a single qubit, defining the encoding feature map as:

\[\phi:i\rightarrow \left|i\right>\]

Therefore, the kernel defined by the inner product is:

\[k(i, j) = \left<\phi(i)|\phi(j)\right> = \left<i|j\right> = \delta_{ij}\]

With \(\delta\) being the Kronecker delta.

The encoding also permits encoding an entire dataset of binary strings \(\mathcal{D}=\{\boldsymbol{x}^1,...,\boldsymbol{x}^m\}\) together as:

\[\left|\mathcal{D}\right> = \frac{1}{\sqrt{M}}\sum_{m=1}^{M} \left|\boldsymbol{x}^m\right>\]

encoding(x: Union[int, numpy.ndarray]) → numpy.ndarray¶

Application of basis encoding to the input.

In basis encoding, each classical bit is mapped into a qubit.

Parameters

x (int or numpy.ndarray) – Either a single sample (int) or a dataset of shape (n_samples,).

Returns

Amplitude vector describing the input encoded into a quantum state. If a dataset is provided, the quantum state will be a superposition of the encodings of every sample in the dataset.

Return type

numpy.ndarray

Raises

ValueError – When an invalid input type is provided.

Examples

>>> a = 1
>>> BasisEncoding().encoding(a)
array([0., 1.])

>>> a = np.array([0, 1])
>>> BasisEncoding().encoding(a)
array([0.70710678, 0.70710678])
