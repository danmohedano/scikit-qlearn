Encoding¶

class skqlearn.encoding.Encoding¶

Bases: abc.ABC

Abstract class to define a data encoding method.

Each implementation must define the encoding method as a feature map \(\phi : \mathcal{X} \rightarrow \mathcal{F}\) taking a sample \(x\) from the input space \(\mathcal{X}\) to the feature space \(\mathcal{F}\).
