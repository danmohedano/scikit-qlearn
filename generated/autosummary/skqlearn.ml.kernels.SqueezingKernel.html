SqueezingKernel¶

class skqlearn.ml.kernels.SqueezingKernel(c: float)¶

Bases: object

Kernel inspired by squeezed vacuum states [SK19]

It is based on the definition of a squeezed vacuum state of the electromagnetic field as:

\[\left|z\right> = \frac{1}{\sqrt{\cosh(r)}} \sum_{n=0}^{\infty}\frac{\sqrt{(2n)!}}{2^n n!} (-e^{i\varphi}\tanh(r))^n\left|2n\right>\]

The notation \(\left|z\right> = \left|(r, \varphi)\right>\) is also used, where \(z=re^{i\varphi}\).

\(x\rightarrow\left|\phi(x)\right>=\left|(c,x)\right>\) can be interpreted as a feature map for \(x\in\mathbb{R}\). The hyperparameter \(c\) determines the strength of the squeezing. This can be extended to vectors of the form \(\boldsymbol{x}=(x_1, ..., x_N)^T\in\mathbb{R}^N\), defining the feature map as:

\[\phi: \boldsymbol{x} \rightarrow \left|(c,\boldsymbol{x})\right>= \left|(c,x_1)\right> \bigotimes ... \bigotimes\left|(c,x_N)\right>\]

It therefore defines the following kernel:

\[k(\boldsymbol{x}, \boldsymbol{x}')=\prod_{i=1}^{N} \left<(c,x_i)| (c, x_i')\right>=\prod_{i=1}^N\sqrt{\frac{\text{sech }c\text{ sech }c} {1-e^{i(x_i'-x_i)}\tanh c \tanh c}}\]

This can be evaluated classically by taking the absolute square of the inner product.
