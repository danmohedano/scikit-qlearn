Bases: object

Interactive Bloch Sphere class

Created to help visualize the transformations of quantum states in the Bloch sphere representation. Allows the user to change the values of \(\theta\) and \(\phi\) for:

\[\left|\psi\right> = \cos\frac{\theta}{2}\left|0\right> + e^{i\phi} \sin \frac{\theta}{2}\left|1\right>\]
