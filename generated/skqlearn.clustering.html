Clustering¶

This module contains implementations of basic distance-based clustering algorithms with the objective of illustrating and comparing classical distance calculation with quantum distance estimation.

skqlearn.clustering.GenericClustering(…)

Generic clustering algorithm used as base for K-Means and K-Medians.

skqlearn.clustering.KMeans(n_clusters, …)

K-Means clustering algorithm based on the generic clustering algorithm structure.

skqlearn.clustering.KMedians(n_clusters, …)

K-Means clustering algorithm based on the generic clustering algorithm structure.
