Utils¶

This module contains utility functions and classes that support the rest of the package.

Job handling

In order to facilitate the configuration of the quantum backends where the quantum subroutines are execute, a singleton class has been implemented. This must be configured before trying to perform any quantum computation.

skqlearn.utils.JobHandler()

Singleton class to contain the configuration for quantum executions.
